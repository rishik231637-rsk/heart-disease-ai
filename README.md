# ❤️ HeartGuard AI Predictor

A machine learning web application that predicts the likelihood of heart disease using patient medical data.

## 🚀 Features
- **AI-Powered:** Uses a Random Forest Classifier trained on the UCI Heart Disease dataset.
- **Interactive UI:** Built with Streamlit for a clean, responsive user dashboard.
- **Real-time Analysis:** Provides a risk probability percentage and visual charts.

## 🛠️ Tech Stack
- **Language:** Python 3.13
- **Framework:** Streamlit
- **ML Libraries:** Scikit-Learn, Pandas, NumPy
